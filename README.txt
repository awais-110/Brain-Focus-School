Brain Focus School Website Prototype
===================================

Files included:
- index.html
- style.css
- script.js
- assets/ folder with logo and gallery images

How to run:
1. Extract the ZIP file.
2. Open index.html in any browser.
3. For hosting, upload all files/folders to your hosting or deploy as a static website.

Important:
- Contact form is a prototype. It opens WhatsApp/email with the entered inquiry.
- Real backend/admin panel can be added later for admissions, notices, gallery upload, and parent messages.
- Logo transparent PNG is included as assets/bfs-logo-transparent.png
